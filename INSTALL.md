Install instructions
--------------------

You can choose between different installation methods.

### Classic method ###

- Install as usual:

        $ make
        # make install

- Uninstall:

        # make uninstall

        
### Arch Linux packgage ###

- Install:

        $ make pkg
        # pacman -U snapman-*.pkg.xz

- Uninstall:
        
        # pacman -Rsc snapman
